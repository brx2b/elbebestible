# Bienvenido a El bebestible

**¿Qué encontrarás?**

En nuestro sitio podrás explorar un catálogo moderno de bebestibles con envíos a todo Chile.
Rápido de navegar, fácil de comprar y mejor aún de beber.

La plataforma está diseñada para ser completamente responsiva y funcionar sin problemas en cualquier dispositivo, aprovechando tecnologías actuales del ecosistema web. <br>
**¡Fácil de comprar y de beber!**

No te pierdas nuestra página web totalmente optimizada y adaptable a tu dispositivo gracias a técnologias modernas para la creación de sitios web.

# Objetivo

Entregar una experiencia de compra simple, ordenada y eficiente, donde un usuario pueda registrarse, iniciar sesión, explorar productos y gestionar su carrito sin fricciones.
El sistema además incorpora una sección administrativa que permite manejar usuarios y bebidas directamente desde la interfaz.

# Detalles técnicos

 - Frontend — React + Bootstrap
   - Estilos con Bootstrap 5 y componentes adaptables
   - Vercel para el desplegue
   - Jest para testing
   - Visual studio code
   - Navegador brave

    
   
 - Backend — Spring Boot
   - Configuración CORS personalizada para permitir acceso
   - Base de datos — PostgreSQ (alwaysdata)
   - Mockito y Swagger para pruebas
   - Intellij community
